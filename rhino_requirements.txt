# Rhino Python Requirements

# These packages are required for Rhino Python scripting
# Note: rhinoscriptsyntax, scriptcontext, and Rhino are not available on PyPI
# They are part of the Rhino3D software installation

# To use these packages, you need to:
# 1. Install Rhino3D software (https://www.rhino3d.com/)
# 2. Use the Python interpreter that comes with Rhino (IronPython)
# OR
# 3. Install the rhinocommon package which provides Python bindings for Rhino

# For standard Python (CPython), you can use:
rhinocommon==7.13.22039.13001  # Use the version that matches your Rhino installation
